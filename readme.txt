== Description ==

Maid Booking plugin adds a simple maid booking slider, which you can add to your post, page or template using [maid_booking] shortcode.

* Send details of booking order to email
* Set base price, price per bedroom/bathroom
* Limit number of bedroom's/bathroom's
* Maid Booking Slider works with half bathroom's.
* Set title, subtitle and button text for the Maid Booking Slider

== Installation ==

* Upload the entire maid-booking folder to the /wp-content/plugins/ directory.
* Activate the plugin through the 'Plugins' menu in WordPress.

You will find 'Maid booking' menu in your WordPress admin panel under Settings menu.
